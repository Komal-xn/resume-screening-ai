# Resume Screening AI

This project builds an AI system that automatically ranks resumes based on how well they match a job description.

## Features
- Resume text processing
- TF-IDF vectorization
- Cosine similarity matching
- Candidate ranking

## Tech Stack
- Python
- NLP
- Scikit-learn
- Machine Learning
